Data is only useful if you trust it. We clean, structure, and analyze data
so the answer you get out the other end is one you can act on - and defend
if someone asks how you got there.

## Who this is for

Teams sitting on data they haven't had time to properly analyze, and
researchers who need statistical rigor applied to their dataset.

## How we work

We start with an honest audit of the data - what's missing, what's messy,
what's usable - before running any analysis. Every deliverable includes a
plain-language writeup of methodology alongside the results.
